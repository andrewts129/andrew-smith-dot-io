  • @@DonaldTrumBot

    A Twitter bot written in Python that uses Markov chains to create (mostly-nonsensical) tweets resembling those of Donald Trump. The account can be found here and the source code is hosted on GitHub .

  • Whispering Pablo

    A "social network" demo with a live feed of messages that are submitted anonymously and vanish after one hour. Under the hood, it's a RESTful web service built with Scala, the Play Framework, and a PostgreSQL database hosted on Heroku. The actual site can be found here and the source code is hosted on GitHub.

  • This Website

    I use this site as a way to learn about various web development languages, frameworks, and tools. Right now, the back end is written in Scala using the Play Framework, the front end is written using plain JavaScript, HTML, and SASS, and it's hosted in a Docker container running on the cloud platform Heroku. The source code can be found on GitHub.

  • Biggest One Hit Wonders Analysis

    An attempt to quantify and rank the biggest one hit wonders, where a "big" one hit wonder is considered to be an artist who found a lot of success with one song, and very little otherwise.

    Data was collected, cleaned, and analyzed in R while D3.js was used for the visualization. The source code for both the R script and the visualization can be found on GitHub .
